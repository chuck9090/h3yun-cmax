# H3Yun CMax 插件安装指南

## 快速安装(推荐)

### 步骤 1: 找到 .vsix 文件

在项目根目录下找到 `h3yun-cmax-0.0.1.vsix` 文件

### 步骤 2: 在 VSCode 中安装

1. 打开 VSCode
2. 点击左侧活动栏的 **扩展** 图标(或按快捷键 `Ctrl+Shift+X`)
3. 在扩展面板右上角,点击 **"..."** (更多操作)按钮
4. 选择 **"从 VSIX 安装..."**
5. 在文件选择对话框中,找到并选择 `h3yun-cmax-0.0.1.vsix` 文件
6. 等待安装完成
7. **重启 VSCode**(重要!)

### 步骤 3: 验证安装和使用

重启后,您可以通过以下**任意一种方式**使用插件:

#### 方式一: 命令面板(推荐)
1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 "氚云" 或 "从氚云构建项目"
3. 选择对应命令即可使用

#### 方式二: 资源管理器右键菜单
- **在空白处或文件上右键**: 显示 "从氚云构建项目"
- **在文件夹上右键**: 显示 "从氚云同步"

#### 方式三: 编辑器右键菜单
- 在任何编辑器中右键,可以选择 "从氚云构建项目"

## 首次使用前准备:获取 h3_token

在使用插件前,您需要先从浏览器获取氚云的认证 Token:

1. 在浏览器中登录氚云平台
2. 按 `F12` 打开开发者工具
3. 切换到 **Application**(应用程序) 标签
4. 展开 **Storage**(存储) → **Cookies**
5. 点击氚云平台域名
6. 找到 **h3_token**,复制其值

详细步骤请参考 README.md 中的"首次使用:获取 h3_token"章节。

## 使用方法

### 首次构建项目(命令面板)

1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 "从氚云构建项目" 或 "氚云"
3. 选择对应命令
4. **在弹出的表单窗口中**:
   - 输入氚云应用编码
   - 点击"📖 如何获取 h3_token?"查看详细指南
   - 输入从浏览器获取的 h3_token
   - 点击"开始构建"
5. 插件会自动验证 Token 有效性
6. 验证通过后开始同步,等待完成(右下角会显示进度)

**注意**: 
- 初次构建项目**只能使用命令面板**方式
- Token 会自动保存到应用目录下的 `.h3token`,下次同步时无需重复输入
- 应用目录会自动生成 `.gitignore`,忽略 `.h3token`
- 构建完成后会询问是否自动执行 `git init` 并提交项目文件
- 如果输入错误,表单会显示具体错误信息,方便您修正

### 同步已有项目(右键菜单或命令面板)

**方式一: 右键菜单**
1. 在左侧资源管理器中,右键点击应用文件夹
2. 选择 **"从氚云同步"**
3. 等待同步完成(Token 已自动保存到 `.h3token`,无需再次输入)
4. 同步完成后,插件会询问是否自动提交本次 Git 变更

**方式二: 命令面板**
1. 按 `Ctrl+Shift+P` 打开命令面板
2. 输入 "从氚云同步"
3. 选择对应命令

### Token 失效处理

如果同步时提示 Token 失效:
1. 插件会弹出提示
2. 点击"重新输入 Token"
3. 输入新的 h3_token
4. 插件会自动继续同步
5. 新 Token 会自动保存到 `.h3token`

如果项目来自 GitHub clone,缺少 `.h3token`,同步时插件会提示输入 Token,并自动重新生成 `.h3token`。

### 代码冲突处理

同步时,插件会智能处理代码差异:

**字段表 (fields.md)**:
- 直接覆盖,无需确认

**代码文件**:
- 自动检测本地和远程的差异
- 发现差异时弹出对话框让您选择:
  - 全部使用本地版本(保留我的修改)
  - 全部使用远程版本(覆盖为氚云版本)
  - 逐个询问(每个文件单独选择)

详细策略请参考 README.md 中的"智能同步策略"章节。

## 常见问题

### Q: 安装后看不到菜单项?

A: VSCode 插件**不能在"文件"菜单栏添加菜单**,但您可以通过以下方式访问:

1. **命令面板**(最可靠): 按 `Ctrl+Shift+P`,输入 "从氚云构建项目"
2. **右键菜单**: 
   - 在资源管理器空白处或文件上右键 → "从氚云构建项目"
   - 在文件夹上右键 → "从氚云同步"
3. **编辑器右键**: 在任何代码编辑器中右键 → "从氚云构建项目"

如果以上方式都看不到,请检查:
- 是否已重启 VSCode
- 是否打开了一个工作区文件夹(必须打开文件夹才能使用)

### Q: 如何获取 h3_token?

A: 
1. 在浏览器中登录氚云平台
2. 按 F12 打开开发者工具
3. 切换到 Application/存储 标签
4. 找到 Cookies 中的 h3_token
5. 复制其值

### Q: Token 会过期吗?

A: 是的,Token 有有效期。过期后:
- 插件会检测到认证失败
- 提示您重新输入 Token
- 输入新 Token 后自动继续同步
- 新 Token 会自动保存

### Q: 点击命令后没有反应?

A: 可能的原因:
- 网络连接问题,无法访问氚云平台 API
- 应用编码输入错误
- Token 无效或已过期
- 氚云接口响应结构变化,需要调整插件内对应解析器

### Q: Token 保存安全吗?

A: 
- Token 以明文形式保存在 `.h3token` 文件中
- 插件会自动将 `.h3token` 写入应用目录的 `.gitignore`
- `cmax.json` 不保存 Token,用于记录应用和表单结构

### Q: 如何卸载插件?

A: 
1. 打开扩展面板(`Ctrl+Shift+X`)
2. 搜索 "H3Yun CMax"
3. 点击插件右侧的齿轮图标
4. 选择 "卸载"
