🔑 如何获取 h3_token

从浏览器 Cookie 中复制氚云认证 Token

1 登录氚云平台
  1. 打开浏览器(推荐使用 Chrome 或 Edge)
  2. 访问氚云平台并登录您的账号
  3. 确保已成功进入氚云工作台
2 打开开发者工具
  1. 在氚云页面按 F12 键
  2. 或者右键点击页面任意位置,选择"检查"
  3. 开发者工具面板会在右侧或底部打开
💡 提示: 如果按 F12 没反应,可以尝试 Ctrl+Shift+I (Windows) 或 Cmd+Option+I (Mac)
3 查找 h3_token
Chrome / Edge
Firefox
  1. 切换到 Application(应用程序) 标签
  2. 在左侧菜单展开 Storage(存储) → Cookies
  3. 点击氚云平台的域名: www.h3yun.com
  4. 在右侧列表中找到 h3_token
  5. 复制其 Value(值) 列的完整内容
💡 提示: h3_token 的值通常是一串较长的字母数字组合,请完整复制
4 粘贴到插件
  1. 回到 VSCode 的输入框
  2. 将复制的 Token 粘贴进去
  3. 点击确定完成输入
⚠️ 注意: Token 有一定有效期,如果后续同步时提示失效,请重新获取新的 Token
