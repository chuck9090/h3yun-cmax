🔑 如何获取构建参数

从氚云构建项目需要填写 AppCode、enginecode 和 h3_token

AppCode 是氚云应用编码。插件使用它定位要构建的应用，它不是应用名称。
1 打开应用信息
  1. 登录氚云平台并进入工作台。
  2. 找到需要构建到本地的应用。
  3. 点击该应用的 应用重命名 入口。
  4. 系统会弹出应用信息框。
2 复制应用编码
  1. 在应用信息框中找到灰色框内容。
  2. 选中灰色框中的完整编码并复制。
  3. 回到 VSCode，粘贴到构建表单的 应用编码 输入框。
填写位置
从氚云构建项目表单中的 应用编码。
提示: 请复制编码值，不要填写应用显示名称。
enginecode 是企业引擎编码。氚云接口请求现在要求在 Headers 中携带该参数。
1 进入系统集成
  1. 使用 系统管理员 账号登录氚云平台。
  2. 点击页面右上角头像。
  3. 进入 系统管理。
  4. 打开 系统集成。
提示: 只有系统管理员可以进入系统管理界面。如果看不到入口，请联系企业管理员获取 enginecode。
2 复制 EngineCode
  1. 在系统集成页面找到 EngineCode 或企业引擎编码。
  2. 复制完整内容。
  3. 回到 VSCode，粘贴到构建表单的 企业引擎编码 enginecode 输入框。
填写位置
从氚云构建项目表单中的 企业引擎编码 enginecode。
注意: EngineCode 和 EngineSecret 都属于重要信息。插件只需要填写 EngineCode，不需要填写 EngineSecret。
h3_token 是当前浏览器登录氚云后的认证 Token。插件会把它保存到本地 .h3token，并自动加入 .gitignore。
1 登录氚云平台
  1. 打开浏览器，推荐使用 Chrome 或 Edge。
  2. 访问氚云平台并登录您的账号。
  3. 确保已成功进入氚云工作台。
2 打开开发者工具
  1. 在氚云页面按 F12 键。
  2. 或者右键点击页面任意位置，选择“检查”。
  3. 开发者工具面板会在右侧或底部打开。
提示: 如果按 F12 没反应，可以尝试 Ctrl+Shift+I 或 Cmd+Option+I。
3 查找 h3_token
Chrome / Edge
Firefox
  1. 切换到 Application 或 Storage 标签。
  2. 在左侧菜单展开 Storage → Cookies。
  3. 点击氚云平台的域名：www.h3yun.com。
  4. 在右侧列表中找到 h3_token。
  5. 复制其 Value 列的完整内容。
提示: h3_token 的值通常是一串较长的字母数字组合，请完整复制。
4 粘贴到插件
  1. 回到 VSCode 的构建表单。
  2. 将复制的 Token 粘贴到 h3_token 输入框。
  3. 点击 开始构建。
注意: Token 有一定有效期。如果后续同步时提示失效，请重新获取新的 Token。
